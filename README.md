## 功能

- [X] 登记与修改作业
- [X] 富文本支持
- [X] 按科目分组作业
- [X] 为作业添加标签
- [X] 主界面全局缩放
- [X] 自动清理过期作业
- [X] 导出作业截图
- [ ] 作业历史记录
- [ ] 插入与管理表情包
- [ ] 插入图片
- [ ] 插入链接
